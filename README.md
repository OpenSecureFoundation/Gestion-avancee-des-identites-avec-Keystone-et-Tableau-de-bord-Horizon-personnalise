# Gestion avancée des identités avec OpenStack Keystone et Tableau de bord Horizon personnalisé

Projet de gestion avancée des identités avec OpenStack Keystone et Tableau de bord Horizon personnalisé

# Objectifs:

• Implémenter un système d’authentification multi-facteurs (MFA) et
fédérée (SAML, OAuth2)

• Étendre Keystone pour supporter la gestion granulaire des rôles et permissions

• Créer des plugins Horizon pour ajouter des fonctionnalités spécifiques à une organisation (ex : suivi budgétaire, rapports sécurité)

• Modifier l’interface pour simplifier la gestion des utilisateurs et projets
